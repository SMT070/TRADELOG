Setup Instructions
~5 minutes
1
Download the EA file
Click the button below to download TradeLog_Sync.mq5 (v1.4) — this is the source file. New in v1.4: Deposits & withdrawals are automatically detected and shown as Account Events — they appear in your trade history but are excluded from all statistics. Open it in MetaEditor and press F7 to compile (takes 2 seconds, produces 0 errors).
Download TradeLog_Sync.mq5
2
Place the file in MT5 and compile
In MetaTrader 5, click File → Open Data Folder from the top menu bar.
A folder window will open. Navigate into MQL5 → Experts
Copy and paste the downloaded TradeLog_Sync.mq5 file into that folder.

Back in MT5, open MetaEditor (press F4), open the file, then press F7 to compile — you should see 0 errors. Then in the Navigator panel, right-click Expert Advisors → Refresh and TradeLog_Sync will appear.
3
Configure WebRequest permission in MT5
Go to Tools → Options from the MT5 top menu, then click the Expert Advisors tab.

In that tab, make sure the settings are as follows:
Allow WebRequests for listed URLs — must be checked / enabled
Allow automated trading — must be unchecked / disabled in Options (the EA only sends data, it does not trade)
In the URL list field below that checkbox, add this exact URL:
https://yuqfoasfefldenkvwyrm.supabase.co/functions/v1/smart-worker
⚠️ The URL must be added exactly as shown above. Then click OK to save and close Options.
4
Attach the EA to a chart and enter your API Key
Open any chart in MT5 (e.g. EURUSD H1 — the symbol does not matter).
In the Navigator panel, find Expert Advisors → TradeLog_Sync and double-click it (or drag it onto the chart).

A settings dialog will open. Click the Inputs tab and fill in:
ApiKey
Paste your API key from the top of this page
SyncIntervalMinutes
Leave as 5 (syncs every 5 minutes)
Loading...
Click OK to attach the EA. You will see a small smiley face icon 🙂 appear in the top-right corner of the chart — this means the EA is running successfully.
✓
You're live! 🎉
The EA will immediately sync your full trade history on first run, then continue syncing every 5 minutes automatically.

To confirm everything is working, look at the Experts tab at the bottom of MT5. You should see a message like:
TradeLog: Sync OK — 12 trade(s), 2 balance event(s) sent.
💡 Keep MT5 open with the chart running for continuous syncing. The EA only runs while MT5 is open.
Troubleshooting
❓ EA says "WebRequest blocked" in Experts tab
Go to Tools → Options → Expert Advisors, make sure Allow WebRequests for listed URLs is checked, and the exact URL https://yuqfoasfefldenkvwyrm.supabase.co/functions/v1/smart-worker is in the list. Click OK and try again.
❓ Trades are not appearing in TradeLog
Check the Experts tab in MT5 for any error messages. Make sure the chart the EA is attached to is still open. Also confirm your API key was pasted correctly in the EA Inputs — it must match exactly what is shown above.
❓ I see a sad face ☹️ instead of smiley on the chart
This means the EA is attached but has an error. Check the Experts tab at the bottom of MT5 to see the exact error. Usually it is a wrong API key or WebRequest not allowed.
❓ EA disappears when I restart MT5
Right-click the chart and click Save as Template. Next time MT5 opens, apply the template to the chart and the EA will be back with all your settings.
❓ I want to re-sync my full trade history
Remove the EA from the chart (right-click → Remove Expert Advisor), then drag it back on and attach it again. It will do a full history sync on first run automatically.
