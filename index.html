Everything you need to know about TradeLog
🚀 Getting Started
Log trades manually using + Add Trade, or import directly from MT5, cTrader, or TradingView using a CSV file. Set your account balance in Settings so your return % is calculated correctly.

☁️ Your Data & Sync
All trades are synced to the cloud in real time — your data is safe across devices. TradeLog auto-syncs every 30 seconds and whenever you return to the app. Use Export CSV from the Import page for a local backup anytime.
